Sr No. Tag name Purpose
1. HTML The begining element for a html page
2. DOCTYPE Tells the browser the document type belongs to
3. Title Displays the title of html page on browser window
4. Meta Tells more information about the html page. Charset defines the encoding of the html page. Name and content attributes can be used together to provide document metadata in terms of name value pairs.
5. Heading Tags from h1 to h6 The heading tags shows the headings within an html documents. Tags h1 to h6 varies in size with h1 being the largest in size.
6. head this is the head section of the html .usually script and css files are included in this section.
7. Body This defines the body section of an html document
8. Table Defines an table in the html document
9. tr defines the row in the html table
10. thead defines the heading section of the html table
11. th defines the the heading of an html table
12. tbody Defines the table body of html table
13. td Defines a column of the html table
